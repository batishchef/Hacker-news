## Инструкция по запуску
1. Чтобы запустить приложение, введите
`yarn install`
`yarn start`
2. Перейдите по адресу localhost:3000

## Технологии
- React
- Typescript
- VKUI
- vk-mini-apps-router
- vite
- Feature-sliced design
